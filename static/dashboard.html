⚡ macOS 一键部署
1

设置 API Key

在「API 配置」标签页填入 DeepSeek 或 OpenAI 的 API Key

2

设置全局环境变量

让 Claude Science 从 Dock/Spotlight 启动时也能读取 ANTHROPIC_BASE_URL

3

安装后台服务

将 proxy 安装为 launchd 服务，开机自启，无需手动启动

4

刷新 OAuth Token

重新生成假 OAuth token（更换账号或 token 过期后需要）

📋 手动启动

复制以下命令在终端执行，即可使用 Claude Science：

export ANTHROPIC_BASE_URL="http://127.0.0.1:9876"
open -a "Claude Science"
